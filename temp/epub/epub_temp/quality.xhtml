Quality Evaluation

This evaluation is based on the rubric located in Appendix A of Developing Quality Technical Information.

Task Orientation

Although the site does provide a great deal of information, it failed to include any task topics. The site does not inform the user how to become an involved in InclusiveVT. It would have been helpful if there was a task topic titled "How to Become Involved". Although InclusiveVT is meant to be mainly informative, a few task topics would have been helpful.

Accuracy

The subjects seem to be well-researched, accurate and informative.

Completeness

As a whole, the site appears to be complete, but it could be improved with task topics. Currently, the site mainly consists of information about events and concepts, Tasks would make the site more complete by allowing the user to use the site based on their needs and interests.

Clarity

The content is concise and easy to understand.

Concreteness

The language is appropriate for the targeted audiences.

Style

Some of the content is a bit dry, but overall is appropriate for the audience.The overall tone of the site is informative yet casual, and captures the essence of InclusiveVT well.

Organization

The site is organized very well and is easy for the user to navigate. The use of Navtitle is appropriate and effective.

Retrievability

Searching for specific information is easy to do on the site.

Visual Effectiveness

The site could improve as far as visuals. Include more images to appeal to the user.
